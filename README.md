## üì¶ Installation

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge recommended)
- [OpenRouter API Key](https://openrouter.ai/keys) (free tier available)

### Quick Start

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Adil-Ijaz7/Aiora_Voice-Assistance.git
   cd Aiora_Voice-Assistance
   ```

2. **Open in Browser**
   - Double-click `index.html`, or
   - Use Live Server extension in VS Code

3. **Configure API Key** (Optional but recommended)
   - Click the ‚öôÔ∏è settings icon
   - Enter your OpenRouter API key
   - Click "Save Settings"

## üéØ Usage

### Voice Commands

| Command | Action |
|---------|--------|
| "Hello" / "Hey" | Greet AIORA |
| "What's the time" | Get current time |
| "What's the date" | Get current date |
| "Open YouTube" | Opens YouTube |
| "Open Spotify" | Opens Spotify |
| "Open Google" | Opens Google |
| "Play [song name]" | Search song on Spotify |
| "Search for [query]" | Google search |
| "Who are you" | Learn about AIORA |
| "Who developed you" | About the developer |

### Chat Interface

1. **Type a Message**: Enter text in the input field and press Enter or click Send
2. **Voice Input**: Click the microphone button to speak your message
3. **Quick Actions**: Use the quick action buttons for common tasks
4. **Clear Chat**: Click the trash icon to clear conversation history
